### 4. Contact Information
<img width="367" height="650" alt="image" src="https://github.com/user-attachments/assets/6b6e5474-1a91-4115-8210-c24a3e519237" />

Once a face is successfully recognized, the application automatically retrieves the person's contact details from the SQLite database — including name, phone number, email, and the recognition confidence level. This demonstrates how facial recognition can be applied to contact management, giving users quick access to key details after a successful match.
